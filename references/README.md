# References

External knowledge the assistant can lean on.

- `sops/` — standard operating procedures. How you do recurring things, written down once.
- `examples/` — output samples and voice references. Real examples of work you liked, so the assistant can match them.

Add files here as new SOPs or style references emerge. A skill can point at a file here (e.g. a voice reference) so its output stays anchored to a real sample.
